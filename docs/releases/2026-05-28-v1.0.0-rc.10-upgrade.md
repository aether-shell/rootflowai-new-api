## Local Patches Retained

### 1. Zero-completion abnormal stream billing suppression

Files:

- `service/text_quota.go`
- `service/text_quota_test.go`

Behavior kept in this release:

- When a text stream ends abnormally without a normal completion and `completion_tokens=0`, the calculated quota is suppressed to `0`.
- The consume log is still recorded for audit.
- The consume log includes:
  - `quota_suppressed=true`
  - `suppressed_quota=<original calculated quota>`
  - `suppress_reason=zero_completion_stream_<end_reason>`

### 2. Disable public image publishing workflows

Files:

- `.github/workflows/docker-build.yml`
- `.github/workflows/docker-image-alpha.yml`
- `.github/workflows/docker-image-nightly.yml`

These local workflow changes are preserved so RootFlowAI does not publish public images by accident during the upgrade.
